# Fitman

## firebase token manager

# how to use
```
[sub commands]
// create .fitman directory & get idToken
fitman init


// show idToken (after init) 
fitman get

// show help
fitman help

// show version
fitman version

[option]
fitman -v get
{
  "access_token": "dummy",

  "token_type": "Bearer",
  "refresh_token": "dummy",
```
